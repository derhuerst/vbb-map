# vbb-map

**Render VBB lines on a map.**

[![chat on gitter](https://badges.gitter.im/derhuerst.svg)](https://gitter.im/derhuerst)


## Installing

```shell
git clone https://github.com/derhuerst/vbb-map.git
cd vbb-map
npm i
```


## Usage

```shell
node render
```
